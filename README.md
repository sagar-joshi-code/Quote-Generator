# Quote Generator App

A simple web application that fetches and displays random quotes using a public API. Click the button to get a new quote along with the author.

---

## Features

- Fetches **random quotes** from `dummyjson` API.
- Displays **quote** and **author**.
- Shows a **loading indicator** while fetching a quote.
- Delays author display for **2 seconds** for better effect.
- Handles API errors gracefully.

---

## Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- Fetch API

---

## How to Use

1. Open `index.html` in a web browser.
2. Click the **"Generate Quote"** button.
3. The quote will appear in the **Quote** section and the author in the **Author** section.
